Довідник шахових дебютів

Питання "Який дебют мені обрати, щоб отримувати хороші позиції в шахових партіях?" бентежить як початківців, тик і професіоналів гри в шахи. Я пропоную сервіс, що допоможе гравцям будь-якого рівня знаходити для себе дебюти, які є надійними, практичними і головне: такими, що підходять саме для їхнього стиля гри. Ця розробка особливо актуальна також для тих хто вже ознайомлений з дебютами, але хоче їх грунтовніше вивчити.

Мета: написання сайту з каталогом дебютів для гри в шахи. Довавання до сайту відповідних елементів для зручного знаходження, зберігання та вивчення дебютів. Актуалізація данних для зручності користувача.

Об'єктом дослідження є: пошукова система заточена на шахові дебюти.

Предметом дослідження є: аналіз роботи існуючих шахових довідників.

Постановка задачі: розробка веб-додатку, з допомогою якого користувач знаходить перелік шахових дебютів, отримує можливість їх зберігати та вивчати. Розробка алгоритмів зображення та вивчення шахових ходів. Додавання гнучкої системи данних для постійного їх оновлення.
